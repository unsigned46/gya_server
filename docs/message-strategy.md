# Message Strategy

이 봇의 메시지는 사용자를 몰아세우기보다 시작 행동을 작게 만들기 위한 장치다.
기본 원칙은 다음과 같다.

- 수치심보다 실행 단서를 준다.
- 목표 전체보다 첫 행동 하나를 말한다.
- "해야 한다"보다 "지금 할 수 있는 가장 작은 행동"을 제안한다.
- pass는 실패가 아니라 의식적 선택으로 다룬다.
- 반복 빈도는 압박보다 재착수 신호로 작동해야 한다.

## Cadence

| 구간 | 기본값 | 의도 |
| --- | --- | --- |
| 시작 직후 | 즉시 1회 | 시간 단서와 첫 행동 제시 |
| Phase one | 2분 간격 | 생각을 줄이고 작은 착수를 유도 |
| Escalation | 6분 후 | 지연을 명확히 인식시키되 비난하지 않음 |
| Phase two | 3분 간격 | 환경 전환, 2분 착수, 의식적 선택 유도 |
| Stop | 15분 후 | 세션을 닫고 다음 시작을 작게 재설계 |

## Tone Rules

- 좋은 문구: "첫 파일만 열자", "2분만 시작하자", "완성이 아니라 착수다"
- 피할 문구: "실망스럽다", "왜 아직도", "의지가 없다", "계속 미루는 중이다"
- 강한 메시지가 필요하면 인격 평가가 아니라 현재 상태와 다음 행동을 말한다.

## Current Defaults

```env
PHASE_ONE_INTERVAL_MINUTES=2
ESCALATION_AFTER_MINUTES=6

```
